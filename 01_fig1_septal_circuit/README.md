# fig1_septal_circuit

This folder contains MATLAB code for unsupervised classification of medial septum (MS) cells based on electrophysiological firing properties.

`git_01_slp_cluster.m` performs unsupervised cell classification and t-SNE embedding/visualization. Example input data are provided in `git_folder/SLP_input_table.mat`.

Before running the script, please unzip `srinivas.gs_mtools-master` and keep the unzipped folder in this directory. The `CellSorter-master` folder should also be kept in this directory. The script was tested in MATLAB R2022b.
