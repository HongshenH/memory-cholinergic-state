# fig2_state_control

This folder contains example code related to Fig. 2, which examines how optogenetic manipulation of hippocampal feedback regulates its own cholinergic state and network-state dynamics.

* `photometry/`: MATLAB code for processing CA1 acetylcholine fiber-photometry signals.
* `power_spectral_density/`: MATLAB code for LFP power spectral density analysis comparing pre-stimulation and stimulation windows.
* `HMM_state_transition/`: Python and MATLAB code for hidden Markov model (HMM) analysis of state transitions.

Each subfolder contains its own README file with example input data, required dependencies and expected output.
