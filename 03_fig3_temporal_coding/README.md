# 03_fig3_temporal_coding

This folder contains example code related to Fig. 3, which examines how hippocampal feedback regulates temporal coding and population dynamics during behavior.

* `seqNMF/`: MATLAB code for visualizing seqNMF results from hippocampal population activity.
* `manifold_isomap/`: Python code for Isomap analysis of hippocampal population activity and its relationship to spatial position.

Each subfolder contains its own README file with example input data, required dependencies and expected output. Demo input data are stored in the local `git_folder/` within each analysis folder, and example expected results are provided in the corresponding `output/` folder.
