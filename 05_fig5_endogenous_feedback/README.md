# 05_fig5_endogenous_feedback

This folder contains example code related to Fig. 5, which examines endogenous CA2MS activity and its relationship to hippocampal population structure during memory-guided behavior.

* `cross_session_manifold/`: Python code for cross-session manifold analysis of CA2MS population activity across behavioral sessions and zone types.

Each subfolder contains its own README file with example input data, required dependencies and expected output. Demo input data are stored in the local `git_folder/` within each analysis folder, and example expected results are provided in the corresponding `output/` folder.
